#  Medical Cost Prediction using Machine Learning

## Overview

This project predicts individual medical insurance expenses using demographic, lifestyle, and health-related data. Multiple machine learning algorithms were developed and compared to identify the best-performing predictive model.

---

## Dataset

- 25,000 observations
- Demographic information
- Lifestyle indicators
- Health conditions
- Insurance plan information
